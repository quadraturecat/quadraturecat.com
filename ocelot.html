Ocelot

A Flexible PC/104 Form Factor Carrier Board

Overview

Ocelot is a robust and flexible carrier board intended to deploy a Raspberry CM4 or CM5 alongside a MiniPCIe device. Particularly well suited to software-defined radio use, it offers a range of interfaces and a reliable builtin supervisor. It is designed with "careful COTS" part selection in mind, and provides an easy to use USB-C debug and diagnostics port.

Features

  • Raspberry Pi Compute Module 4 or 5
  • MiniPCIe card slot: optimized for use with a LimeSDR-XTRX radio
  • In-situ JTAG interface for LimeSDR-XTRX FPGA debugging
  • MicroSIM card slot
  • MicroSD card slot (for use with CM4/CM5 Lite versions only)
  • Dual MCP2518FD CAN-FD interfaces
  • Multiple UARTs, intended for use across power domains
  • Power domain isolated "control interface" with UARTs and boot signals
  • Dual CSI camera interfaces
  • Capactively-coupled gigabit ethernet interface
  • USB-C power and network connection to compute module, for easy development and provisioning
  • USB-C (5V) or DC (5V to 28V) power inputs
  • "Remove before flight" jumpers to enter USB boot mode or disable hardware watchdog
  • Onboard system supervisor and watchdog
  • PC/104 mechanical form factor, suitable for industrial controls, cubesats, robotics, and more
  • Hirose DF13 connectors for additional interfaces
  • Compatible with a range of common cubesat PC/104 signal assignments
